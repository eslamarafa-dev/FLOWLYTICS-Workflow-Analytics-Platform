## ğŸš€ Getting Started

### Prerequisites

- **Node.js** 16+ (optional, for local server)
- **Modern browser** (Chrome, Firefox, Safari, Edge)

### Installation

#### Option 1: Using Serve (Recommended)

```bash
# Clone the repository
git clone https://github.com/yourusername/flowlytics.git
cd flowlytics

# Start a local server
npx serve

# Or using http-server
npx http-server -p 3000

# Open http://localhost:3000
```

#### Option 2: Using Python

```bash
# Python 3
python -m http.server 3000

# Python 2
python -m SimpleHTTPServer 3000

# Open http://localhost:3000
```

#### Option 3: Using VS Code

1. Install **Live Server** extension
2. Right-click on `index.html`
3. Select **"Open with Live Server"**

### Environment Variables

Create a `.env` file (optional, for future API integration):

```env
# API Configuration
VITE_API_URL=https://api.flowlytics.com
VITE_API_KEY=your_api_key_here

# Feature Flags
VITE_ENABLE_ANALYTICS=true
VITE_ENABLE_DEBUG=false
```

### Development

```bash
# Watch for changes (using serve)
npx serve --watch

# Or use any development server with hot reload
```
